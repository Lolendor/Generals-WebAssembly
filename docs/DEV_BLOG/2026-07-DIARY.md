## 2026-07-22: Localize the WebAssembly launch screen

Added Russian and US English localization to the browser launch screen without
changing the selected game build or the language used by the game itself.

- Added separate `ru.js` and `en.js` translation dictionaries.
- Select the initial shell language from browser preferences and remember a
  manual override in local storage.
- Added a compact language switch using native Russian and United States SVG
  flag assets instead of emoji.
- Localized launch settings, download progress, storage errors, and loader
  status messages.
- Added a footer credit linking to `Lolendor/Generals-WebAssembly`, identifying
  Lolendor as the author and displaying the GPL-3.0 license.

Validation:

- JavaScript syntax checks passed for the shell and both dictionaries.
- HTML and SVG validation passed.
- Automated checks covered browser-language selection, saved overrides, and
